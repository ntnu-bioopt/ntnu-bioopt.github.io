--- layout: page title: Real-time processing software permalink: /software/realtime --- GitHub repositories for the real-time software packages used in A. Bjorgan, L. L. Randeberg, "Towards real-time medical diagnostics using hyperspectral imaging technology", Proc. SPIE 9537 (2015) are listed below:
  • libspectralmask (Removal of non-skin regions after reflectance calibration)
  • libmnf (Noise removal)
  • libgpudm (Estimation of skin optical properties)
  • libhyperwavelet (Wavelet transform)
